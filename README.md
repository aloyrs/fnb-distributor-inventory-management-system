# smart-inventory-management-system

NUS Project for TIC2601 Database and Web Applications

`backend` folder stores backend code (API endpoints) and database configuration

To start backend:

- Have "DB Browser for SQLite" installed.
- Open a terminal `cd backend` and run `npm run dev`:
  - This starts backend application and sequelize ORM will create the database with tables.
  - Also populates data into the database tables.
