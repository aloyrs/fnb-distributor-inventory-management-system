# smart-inventory-management-system

NUS Project for TIC2601 Database and Web Applications

`backend` folder stores backend code (API endpoints) and database configuration

To start backend:

- Have "DB Browser for SQLite" installed.
- Open a terminal `cd backend` and run `npm start` to start the backend application and sequelize ORM will create the database with tables.
- Open another terminal `cd backend` and run `npm run seed` to populate data into the database tables.
